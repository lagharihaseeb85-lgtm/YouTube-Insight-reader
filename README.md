## 🎯 Overview

YouTube Insight Pro ek production-ready web app hai jo kisi bhi YouTube video ka complete AI-powered analysis karta hai. Bas URL paste karo aur ye dashboard ban jayega:

- 📺 Video metadata + channel stats
- 💬 500 comments ka sentiment analysis 
- 🧠 Gemini AI se summary, SEO keywords, growth tips
- 📈 Virality score + engagement metrics
- 📄 Professional PDF report download

## ✨ Features

### 1. 📺 Video Metadata
- Title, Channel, Subscribers, Views, Likes, Comments
- Upload Date, Duration, Thumbnail, Tags
- Channel profile picture

### 2. 💬 Comment Analysis
- Top 100-500 comments fetch
- Sentiment: Positive / Neutral / Negative using VADER
- Top 5 Positive + Top 5 Negative comments highlight
- Word Cloud generation

### 3. 🧠 AI Video Analysis - Gemini 1.5 Flash
- Video Summary
- 5 Key Points
- Topics Covered
- SEO Keywords
- Audience Intent
- Content Category

### 4. 📈 Engagement Insights
- Engagement Rate = (Likes + Comments) / Views * 100
- Like-to-View Ratio
- Comment-to-View Ratio  
- Virality Score out of 100

### 5. 🚀 Growth Suggestions
- Title Optimization
- 3 Thumbnail Ideas
- Content Improvement Tips
- SEO Strategy

### 6. 📄 PDF Report
- One-click downloadable professional PDF
- Includes all metadata, charts, AI insights
- Word cloud included
